PHP MVC Framework - YeahPHP
---

> 说明:该版本为草稿版，仅供学习参考使用，不能用于生产环境

##### 已经实现以下基础组件:

- Ioc         控制反转和依赖注入容器
- Config      配置管理器
- Http        Request 、 Response、Cookie
- Event       事件管理
- Routing     路由管理 
- ClassLoader 类加载器

##### 缺少基础组件

- Mvc       Model、View、Controller
- Log       日志
- Session   会话
- Cache     缓存
- Database  数据库操作
- Cli       命令交互


> session、log、cache、database 组件之前已经实现过，为了符合当前框架架构思路，准备重新编写
